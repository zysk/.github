# Zysk Technologies - Organization GitHub Templates

This repository contains default GitHub templates and configurations that apply to all repositories in the Zysk Technologies organization.

## What's in this repository?

- **Issue Templates**: Standardized templates for bug reports, feature requests, and support questions
- **Pull Request Template**: Standard PR checklist for code reviews
- **Code of Conduct**: Guidelines for respectful collaboration
- **Contributing Guidelines**: How to contribute to Zysk projects
- **Security Policy**: How to report security vulnerabilities

## How does this work?

GitHub automatically applies these templates to any repository in the `zysk` organization that doesn't have its own custom templates.

### For Repository Maintainers

If a specific repository needs custom templates:
1. Create a `.github` folder in that repository
2. Add your custom templates there
3. They will override these organization-wide templates

### For Contributors

When creating issues or pull requests in any Zysk repository, you'll see these templates automatically. Please use them to ensure consistent communication across all projects.
