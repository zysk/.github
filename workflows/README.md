# workflows/ — Reference Copies Only

> **⚠️ These files are reference copies and do NOT run org-wide.**

## Why this folder exists

Files in `workflows/` of the org-level `.github` repo only run **within this repository itself** — they do not automatically apply to other repositories in the Zysk organization.

## Active starter workflows

The live, reusable versions of these workflows are in [`workflow-templates/`](../workflow-templates/). They appear as one-click suggestions in the **Actions → New workflow** tab of any org repo.
